# Testing

## Unit testing

`yarn run test` runs tests with mocha.

`yarn run test:watch` runs tests and watches for changes.

## E2E testing

`yarn run test:e2e:setup` to set up for e2e testing.

`yarn run test:e2e` to run e2e tests using nightwatch.
